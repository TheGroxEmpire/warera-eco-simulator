WarEra Economy Skill Simulator

Simple tool for PP flow, material output, and profit optimization.

Created by GroxEmpire

1. Skill Allocation

Skill level cost is progressive: level 1 costs 1 point, level 2 costs 2 more, level N costs N more. Total cost for level N is 1+2+...+N.

Per-level gains: Energy +10, Entrepreneurship +5, Production +3, Companies +1, Management +2. Max level per skill: 10.

2. Operation Settings

Set general usage % here. Entrepreneurship action planning is configured in the Entrepreneurship Plan section.

3. Company Setup

Each company has its own specialization, AE level, workers, worker energy/production, worker wage, production bonus, and worker fidelity.

4. Entrepreneurship Plan

Plan self-work actions per 10h cycle by assigning each action slot to a company. Each action costs 10 Entrepreneurship. Optimizer can optimize this plan for you.

6. Optimization

Choose what optimizer should change: skill allocation, entrepreneurship plan, or both.

7. Results

Scenario A is active and auto-saved on every change.
